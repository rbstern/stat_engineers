  1. Um sistema tem \(10\) componentes que falham independentemente com probabilidade \(0.1\).

  • Considere que o sistema tem os componentes em série, isto é, ele falha se qualquer um dos componentes falhar. Qual a probabilidade de o sistema falhar?

  • Considere que o sistema tem os componentes em paralelos, isto é, ele falha somente se todos os componentes falharem. Qual a probabilidade de o sistema falhar.

  • Considere que dois sub-sistemas são formados a partir de \(5\) componentes em série. Se o sistema é composto pelos dois sub-sistemas em paralelo, qual é a probabilidade de ele falhar?
