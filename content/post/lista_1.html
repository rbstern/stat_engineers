  1. [@Wickham2016] Para este exercício, carregue o banco de dados “mpg” e seu dicionário:
library(tidyverse)
data(mpg)
help(mpg)

  • Classifique o tipo de cada uma das variáveis no banco de dados.

  • Construa um gráfico de dispersão entre as variáveis “hwy” e “cyl”. Que tipo de associação linear você espera entre estas variáveis? O resultado é compatível com sua intuição?

  • Utilizando o seguinte comando é possível visualizar três variáveis simultaneamente:

ggplot(data = mpg) + 
  geom_point(mapping = aes(x = displ, y = hwy, color = class))

No eixo x observamos a cilindrada do motor, no eixo y o número de milhas percorridas por litro de combustível e a cor indica cada tipo de carro. Interprete o gráfico obtido indicando como o eixo x se relaciona com o eixo y. Existe algum tipo de carro que é diferente das demais em relação a esta relação? Por que você acha que este é o caso?
